# Puma-Data-Challenge
This project was given by a company for an interview


Here is a file containing data collected on the products of the brand Puma.
- clothes_puma.csv contains contains static product data
- docs is a ReadMe
- evolutions_puma.csv includes the evolution of certain values.

The purpose of the box is (i) to identify best sellers within their collection, (ii) to build a predictive model on the best seller attribute.
